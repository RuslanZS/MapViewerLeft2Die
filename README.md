# MapViewerLeft2Die
Моя не удачная попытка модифицировать просмотрщик MapViewer от Left 2 Die (J2ME Iceberg 3D engine)
<br>В планах было реализовать просмотр 3d карт через ФС телефона (JSR75), Fallback для отсутсвии TEXTURE=, Fallback для отсутсвии SKY_TEXTURE=, статичная 3D карта при отсутствии MODEL=, ограниченая в движении камеры статичная 3D карта при отсутствии SKY_MODEL=
<br>Код не в рабочем состоянии....
